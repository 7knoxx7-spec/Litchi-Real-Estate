# ليتشي العقارية – Litchi Real Estate

منصة عقارية إنتاجية كاملة بدون بيانات وهمية، تربط كل صفحة وزر وكارت ببيانات حقيقية أو APIs خارجية. تدعم العربية والإنجليزية.

## القصة والمشكلة
- السكن للشباب والعائلات يتطلب سرعة وثقة وخرائط دقيقة.
- التطبيقات التقليدية تستخدم بيانات غير محدثة أو صفحات فارغة.

## الحل
- منصة إنتاجية تربط العقارات الحقيقية بإحداثيات فعلية، خرائط تفاعلية، مشاركة اجتماعية، توصيات ذكية، ومفضلة للمستخدم.

## السوق
- الإمارات ودول الخليج مع توسع تدريجي حسب مصادر بيانات مفتوحة وتجارية.

## التقنية
- Frontend: React + Vite + TanStack Query + shadcn/ui.
- Backend: Node.js + Express + Prisma (SQLite) مع JWT ورفع ملفات.
- خرائط: Leaflet + OpenStreetMap Tiles، تكامل Mapbox اختياري.

## الداتا
- مصادر خارجية: Nominatim (Geocoding)، Overpass API (OSM)، RapidAPI (عقارات)، Apify/ZylaLabs حسب الترخيص.
- قاعدة البيانات: نماذج User, Property, Booking, Favorite, Review, Analytics, Payment.

## الخريطة
- تفاعلية بالكامل مع علامات العقارات الفعلية، مشاركة الموقع، اتجاهات، ستايلات متعددة.

## قابلية التوسع
- فصل الخدمات، خلفيات jobs عبر BullMQ (جاهز للتكامل)، دعم بوابات دفع Stripe/PayPal.

## لماذا تستثمر الآن
- تجربة كاملة جاهزة، بيانات فعلية، هوية واضحة، قابلية نمو سريعة.
